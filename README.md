# React RecipeApp

Responsive Recipe App made using React and Bootstrap.

<hr />

## What I Used

- React
- React Router Dom
- Styled Component



### Installation Options

```
$ git clone https://github.com/farukkursun/RecipeApp
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/farukkursun/RecipeApp
```

```
$ yarn
```

```
$ yarn start
```
